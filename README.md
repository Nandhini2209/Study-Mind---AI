## 🚀 How to Run (Super Easy — 1 File Only!)

### Option 1 — VS Code (Recommended)
1. Open `index.html` in VS Code
2. Right-click → **"Open with Live Server"**
3. App opens in browser ✅

### Option 2 — Any Browser
1. Just **double-click** `index.html`
2. It opens in your browser directly ✅

### Option 3 — Antigravity / Online IDE
1. Upload `index.html`
2. Click Run or Preview
3. Done ✅

---

## 🔑 Setup API Key (One Time Only)

1. Go to → https://console.anthropic.com
2. Sign up → Click **API Keys** → Create Key
3. Copy the key (starts with `sk-ant-...`)
4. Paste it in the app's **"Enter API Key"** box
5. Click **Connect AI** ✅

> Key is saved in your browser. Never shared anywhere.
